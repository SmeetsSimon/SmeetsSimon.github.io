By Simon Smeets
Category: GIP

Vandaag is het de laatste dag dat mnr Isaris me kan helpen. Ik heb hem gevraagt om me te helpen om grafieken te maken op mijn website. Ik heb een website gevonden die grafieken maakt maar die doen dit met Java script. Hiermee heb ik nog nooit gewerkt dus weet ik ook niet echt hoe ik het moet laten zien in mijn eigen website. Het Java script is correct maar ik moet veel dingen in mijn django project om het te laten zien in mijn website. Mnr isaris heeft me hierbij geholpen zodat ik een grafiek heb van de data die in mijn server staat opgeslagen dit heb ik nu voor temperatuur, luchtdruk en luchtvochtigheid. Ik kan in de host deze waardes veranderen en ze worden dan vertoont in de grafiek op mijn site. Dit is het begin van de data die ik hierin zou moeten opslaan hoe weet ik wel nog niet.
